This is a minimal portfolio scaffold built with React, Vite and Tailwind CSS. It includes a dark/light mode toggle, sticky glass navbar, animated intro, About and Skills sections.

Quick start:

```bash
# install deps
npm install

# dev server
npm run dev
```

Files to protect in `.gitignore`: `node_modules/`, `.env`, `/public/resume.pdf`.

You can customize `src/components` to add projects, contact form, or a real resume PDF at `/public/resume.pdf`.
